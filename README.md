# vk-ytmusic

Экспорт музыки из ВКонтакте в YouTube Music.

Треки, которые есть в YTM, получают лайк. Те, которых нет -- скачиваются из ВК и загружаются в YTM как личные треки (раздел "Загрузки", затем автоматически добавляются в "Мне нравится").

**Единственное требование -- Python 3.8+. Остальное скрипт установит сам.**

---

## Установка и запуск

**1. Скачай скрипт**

```bash
git clone https://github.com/FoxlikeCreature/vk-ytmusic
cd vk-ytmusic
```

**2. Запусти**

Linux:
```bash
python vk_ytmusic.py
```

Windows:
```cmd
python vk_ytmusic.py
```

При первом запуске скрипт:
- Проверит наличие ffmpeg и установит его через менеджер пакетов системы (pacman / apt / dnf / winget и т.д.), если его нет
- Попробует установить browser-cookie3 для автоматического получения куков YouTube
- Спросит логин ВК и страницу с музыкой
- Проведёт авторизацию YouTube Music

Больше ничего делать не нужно.

---

## Если Python не установлен

**Linux:**
```bash
# Arch
sudo pacman -S python

# Ubuntu/Debian
sudo apt install python3

# Fedora
sudo dnf install python3
```

**Windows:** скачай с [python.org](https://www.python.org/downloads/), при установке поставь галку "Add Python to PATH".

## Флаги

```
python vk_ytmusic.py              # обычный запуск
python vk_ytmusic.py --dry-run    # тест без изменений (не лайкает, не загружает)
python vk_ytmusic.py --reset      # сбросить прогресс и обработать всё заново
python vk_ytmusic.py --reauth-ytm # обновить авторизацию YTM (куки протухли)
```

---

## Как работает

1. Скрипт получает список треков со страницы ВКонтакте
2. Каждый трек ищется в YouTube Music по исполнителю и названию
3. Если нашёл -- ставит лайк
4. Если не нашёл -- скачивает MP3 из ВК и загружает в YTM
5. Прогресс сохраняется в `progress.json` -- при прерывании продолжит с того же места

---

## Авторизация ВК

Логин и пароль ВКонтакте. Двухфакторная аутентификация поддерживается.
VK API-токен не нужен.

---

## Авторизация YouTube Music

При первом запуске скрипт предложит выбрать браузер (Chrome с выбором профиля, Firefox, Opera и т.д.) и автоматически возьмёт куки. Если не получится -- попросит выполнить одну команду в консоли браузера.

Куки хранятся локально в `browser.json`. Если протухли -- запусти с `--reauth-ytm`.
